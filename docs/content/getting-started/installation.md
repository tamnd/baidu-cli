## Homebrew

```bash
brew install tamnd/tap/baidu
```

## Pre-built binaries

Download the archive for your platform from
[Releases](https://github.com/tamnd/baidu-cli/releases), extract it, and
place `baidu` on your `$PATH`.

## Go

```bash
go install github.com/tamnd/baidu-cli/cmd/baidu@latest
```

## Docker

```bash
docker run --rm ghcr.io/tamnd/baidu:latest hot
```

## Linux packages

`deb`, `rpm`, and `apk` packages are available on the
[Releases](https://github.com/tamnd/baidu-cli/releases) page.
